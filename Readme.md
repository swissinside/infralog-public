## Supported File Types
- *.bin
- *.dse
- *.ilg/z (proprietary)
- *.jru
- *.mdr_i
- *.tel
- *.tru
- *.usar

## Features
- DMI Icons as images
- Position related to trackside BG-kilometer
- trackside topology as image
